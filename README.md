# Meu Site Estático

Este é um site estático simples criado para ser hospedado no GitHub Pages.

## Como hospedar no GitHub Pages

1. Faça upload deste repositório para o GitHub.
2. Vá para as configurações do repositório (Settings).
3. Role para baixo até "Pages" na seção "Code and automation".
4. Em "Source", selecione "Deploy from a branch".
5. Escolha a branch "main" e a pasta "/ (root)".
6. Clique em "Save".
7. O site estará disponível em `https://<seu-usuario>.github.io/<nome-do-repositorio>/`.

## Personalização

- Edite `index.html` para alterar o conteúdo da página.
- Modifique `styles.css` para ajustar o estilo.
- Adicione mais arquivos HTML, CSS ou JavaScript conforme necessário.

## Solução de Problemas

- Certifique-se de que o arquivo `index.html` está na raiz do repositório.
- Se o site não carregar, verifique se a branch e a pasta estão corretas nas configurações do Pages.
- Para sites sem Jekyll, adicione um arquivo `.nojekyll` na raiz se necessário.
